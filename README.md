# 🏋️ Otthon Fit — 6 hónapos calisthenics terv

Személyre szabott otthoni fitness alkalmazás kezdő calisthenicshez. Push-up bar + kanapé az egyetlen szükséges eszköz.

## Mit tud?

- **26 hetes, 4 fázisú program** — fokozatosan halad a könnyű alapozástól a teljes körű állóképességi edzésig.
- **Edzésnapló** — minden szettet, ismétlést, RPE-t és pihenőidőt ment.
- **Súlykövetés** — napi súlyrögzítés, vizuális grafikon, lineáris cél a 89 → 75 kg úthoz.
- **Hangulat / energia / alvás napló** — látod, mi befolyásolja a teljesítményt.
- **Gyakorlattár** — minden mozgáshoz technika, kulcspontok, könnyítés és nehezítés.
- **Helyi tárolás** — minden adat a böngésződben marad (localStorage). Export / import JSON-ban.

## Indítás helyben

```bash
npm install
npm run dev
```

A `http://localhost:3000` címen érhető el.

## Deploy

A projekt Vercel-re van optimalizálva — push után automatikusan deploy-ol.

## A program szerkezete

- **1. fázis (1–8. hét)**: alapozás, technika, mozgástartomány.
- **2. fázis (9–16. hét)**: erő és izomépítés.
- **3. fázis (17–22. hét)**: intenzitás, tempó, finisherek.
- **4. fázis (23–26. hét)**: csúcs, állóképesség teszt.

Heti 4–5 edzés (push / pull / láb / kondi / opcionális full body) + 1 mobilitás nap + napi séta.
